## 🚀 Fonctionnalités

- 🎶 Song Request via `!sr` ou reward Twitch
- 📋 Ajout automatique dans une playlist YouTube
- 🔄 OAuth Google avec refresh token automatique
- 🎧 Affichage musique actuelle avec `!song`
- ⏭️ Affichage prochaine musique avec `!next`
- 📜 Affichage playlist avec `!playlist`
- 🧹 Clear playlist avec `!clearplaylist`
- 🔐 Setup OAuth via `!godasytm`

---

## ⚡ Installation rapide

1. Télécharger le setup depuis l’onglet **Releases**
2. Installer YouTube Music Desktop
3. Importer le fichier `.sb` dans Streamer.bot
4. Configurer Google Cloud
5. Remplir les Set Arguments dans l’action `godasytm`
6. Lancer dans le chat :

```text
!godasytm
